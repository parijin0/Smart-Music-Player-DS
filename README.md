# Smart Music Player (Data Structures Project)

A console-based Smart Music Player built using C++ to demonstrate the practical application of core Data Structures.

## Features
- Music Library using Hash Maps
- Playlist using Doubly Linked List
- Play History using Stack
- Upcoming Songs using Queue
- Recommendations using Priority Queue
- Song Similarity using Graph + DFS
- Top Rated Songs using Custom Max Heap

## Technologies Used
- C++
- STL (stack, queue, unordered_map, priority_queue, vector)

## How to Run
1. Compile the code:
   g++ main.cpp -o music_player
2. Run:
   ./music_player
